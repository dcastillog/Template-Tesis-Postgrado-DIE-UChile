# Template-Tesis-Postgrado-DIE-UChile
Versión del template umemoria de la FCFM Uchile (https://github.com/dccuchile/memoria-tesis-latex), adaptada a los requerimientos de postgrado del departamento de ingeniería eléctrica (DIE). En esta versión del template se incluye una guía con ejemplos y recomendaciones para la confección de cada apartado del documento.
